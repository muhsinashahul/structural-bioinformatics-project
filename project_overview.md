# Repurposing FDA Approved Drugs to Inhibit Monkeypox DNA Polymerase  
This project involved identifying potential inhibitors for Monkeypox DNA polymerase using  
molecular docking and molecular dnamics simulation approaches (Schrodinger, PyMOL).

## Objectives  
- Select FDA-approved drugs  
- Retrieve target structure
- Prepare target and ligands
- Perform docking  
- Analyze binding affinities
- Evaluate pharmacokinetic properties using ADMET analysis
- Identify top repurposed candidates

## Tools Used  
- Schrodinger   
- ADMETlab  
- PyMOL  
- SwissADME  

## Outcome  
Shortlisted candidate molecules showing promising binding affinity toward Monkeypox DNA polymerase.
